# Seti + Storybook icon theme

This is a local clone of VS Code Seti with one additive change:

- `*.stories.ts`
- `*.stories.tsx`
- `*.stories.js`
- `*.stories.jsx`
- `*.stories.mdx`

All files above use a dedicated Storybook icon.

## Install locally (unpacked)

1. Copy this folder to your local VS Code extensions directory:
   - Linux/macOS: `~/.vscode/extensions/seti-storybook-icons-0.0.1`
   - Windows: `%USERPROFILE%\\.vscode\\extensions\\seti-storybook-icons-0.0.1`
2. Reload VS Code.
3. Run `Preferences: File Icon Theme` and select `Seti + Storybook`.

## Notes

- Base Seti files are from `microsoft/vscode` `extensions/theme-seti`.
- This keeps Seti as-is and only expands Storybook file associations.
