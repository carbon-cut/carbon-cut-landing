# Carbon Cut Stitch Theme

Use this file for Stitch's theme editor or theme context.

This file is only about visual language and theme direction.

It is not the homepage content brief.

## Theme Identity

Carbon Cut should feel:

- credible
- climate-serious
- light
- airy
- distinctive
- modern
- soft but not bland
- less corporate than competitors
- human
- caring
- approachable
- hopeful

Carbon Cut should not feel:

- generic SaaS
- climate consulting
- polished enterprise software
- luxury-branded
- trendy for its own sake
- emotionally flat
- heavy
- dark and institutional
- cold
- intimidating

## Core Visual Goal

The theme should make Carbon Cut feel:

- like a real product with a point of view
- more alive than competitor climate-tech sites
- authored rather than templated
- serious without becoming corporate
- fresh without becoming startup-generic
- human rather than systemic

If the result looks cleaner but also more generic, it is wrong.

## Colors

### Primary

Use a dark teal / green-teal primary similar to the current site, not a deep forest green.

It should carry:

- identity
- trust
- contrast
- recognizability

### Secondary

Use a vivid orange accent similar to the current highlighted headline words.

Use it for:

- emphasis
- selected words
- key highlights
- occasional action emphasis

Do not let the interface become orange-led.

### Tertiary

Avoid introducing extra earthy or brown tertiary tones unless absolutely necessary.

The current identity does not need a heavy tertiary system.

### Neutral

Use very light cool backgrounds close to the current pale blue-gray / icy neutral direction.

Backgrounds should feel:

- soft
- light
- breathable
- open
- welcoming

Avoid:

- beige editorial backgrounds
- expensive cream tones
- dark moody neutrals
- enterprise gray

## Typography

### Headline Font

Headlines should feel:

- serious
- confident
- memorable
- expressive

Do not push typography toward luxury editorial or prestige-consulting aesthetics.

### Body Font

Body text should feel:

- clear
- practical
- modern
- readable

### Label Font

Labels should feel:

- functional
- clean
- understated

## Corner Radius

Use moderate softness.

The interface should feel friendly and usable, not rigid and not overly sculpted.

Avoid:

- sharp institutional corners
- premium-enterprise geometry
- over-designed luxury rounding

## Surface Direction

Use:

- light cool backgrounds
- soft white cards
- gentle contrast between surfaces
- light, airy separation

Avoid:

- glassmorphism
- glossy effects
- heavy drop shadows
- border-heavy component systems
- dark editorial blocks used just to signal seriousness
- beige-on-black prestige contrast systems

## Layout Feel

The theme should support layouts that feel:

- open
- calm
- fresh
- slightly unconventional
- less templated than competitors
- emotionally easy to enter

Do not push the design toward:

- enterprise dashboards
- generic SaaS landing pages
- consulting-site polish
- architectural-editorial corporate mood boards

## Component Feel

Buttons should feel:

- direct
- confident
- simple
- bright enough to create life

Cards should feel:

- useful
- restrained
- secondary to hierarchy and spacing
- light rather than heavy

Navigation should feel:

- calm
- clear
- stable
- not enterprise

Do not use components to make the page feel more corporate or more generic.

## Feeling Check

The theme is correct if it feels:

- recognizably Carbon Cut
- more distinct than competitors
- serious without flattening personality
- lighter and more alive than corporate climate-tech
- human and welcoming without becoming childish

The theme is wrong if it feels:

- safer but more generic
- cleaner but less memorable
- more polished but more corporate
- darker, heavier, or more “premium enterprise” than the current Carbon Cut feeling
- less humane than the original emotional tone
